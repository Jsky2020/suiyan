jQuery 和 HTML DOM

DOM 是 Document Object Model（文档对象模型）的缩写。

HTML DOM 定义了所有 HTML 元素的对象和属性，以及访问它们的方法。 换言之，HTML DOM 是关于如何获取、修改、添加或删除 HTML 元素的标准。

jQuery 是一个 JavaScript 库。 jQuery 极大地简化了 JavaScript 编程。
